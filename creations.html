My Creations

Here you can find my websites, widgets, apps and anything else that I make. Some of the following widgets were made after following a tutorial online. To ensure better understanding I have re-typed and adapted each exercise without following the code directly. I will also link to the tutorial and write a brief overview of things that I have learnt as a blog post.

Odin In Hanoi Website

I wanted to encourage people living in Hanoi to consider joining me in learning web development through the Odin Project. This was the first site that I built after completing the CodeCademy HTML CSS track

Odin in Hanoi #2

I wanted to play around with different styling, having written the content already. This version never went live, but it was very quick to put together. Smooth scrolling between pages could be added.

Snake Game

Following a codeplayer tutorial, I made a copy of the classic snake game, using an HTML canvas element, and JavaScript along with jQuery. It is a great way of putting the JavsaScript I have learnt into practice.

Etch-A-Sketch

Small sketch-pad project. I made this as part of The Odin Project's curriculum to practice the jQuery code learnt from the CodeCademy track. Possible improvements: 3-tone shading and no-colour shading

Calculator

Made after watching the Codeplayer's tutorial. Increased functionality and adapted design are two things I would like to improve.
